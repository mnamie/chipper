# Chipper

Yet another CHIP8 emulator for Linux!

### Instructions:

1. Clone repository
2. Install libsdl2-dev (`sudo apt-get install libsdl2-dev` on Ubuntu, for example)
3. Run `make` in the top directory
4. Start playing with `./chipper 1500 path_to_rom.ch8`
